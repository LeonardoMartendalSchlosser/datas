# datas

Uma biblioteca Python simples e fácil para manipulação de datas, permitindo adicionar dias, calcular diferenças, formatar datas, validar e muito mais.

---

## Funções Disponíveis

-adicionar_dias("2025-03-10", 5)
-subtrair_dias("2025-03-10", 10)
-diferenca_dias("2025-03-10", "2025-03-01")
-eh_final_de_semana("2025-03-15")
-eh_dia_util("2025-03-14")
-formatar_data("2025-03-10", "%d/%m/%Y")
-data_hoje()
-data_valida("2025-02-30")
